# portfolio-optimizer-python

A Python project to simulate and optimize a stock portfolio using historical data, Sharpe ratio, and efficient frontier analysis.

---

## Project Overview

This project simulates a diversified stock portfolio and optimizes it using historical stock price data. It calculates returns, risks, and the Sharpe ratio, and applies efficient frontier analysis to find optimal asset allocations.

---

## Features

- Pulls stock data using `yfinance`
- Calculates portfolio metrics (returns, volatility, Sharpe ratio)
- Visualizes efficient frontier
- Saves result graphs as PNG files in the `figures/` folder

---

## How to Run

1. Open the `portfolio_optimizer.ipynb` notebook in Jupyter
2. Run the cells in order
3. Check the `figures/` folder for the saved visualizations
